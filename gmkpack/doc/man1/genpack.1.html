NAME

genpack − display the genesis of a pack.

SYNOPSIS

genpack

DESCRIPTION

genpack displays on the standard output how a pack has been generated at the time of its creation. The working directory must be the pack main directory.
The information displayed consists in the absolute path of the procedure gmkpack invoked, followed by all the options values explicitely used.
This command is helpful to create new scripts for compilation, libraries archiving and/or links editions with gmkpack upon a pack which already exists, because it ensures the compatibility of the background codes and libraries used.

DIAGNOSTICS

If the working directory is not a pack main directory then the following message is displayed :
"No gmkpack genesis file in this directory."
