NAME

showpack − find the last two versions of source files

SYNOPSIS

showpack file1 [ file2 ] ...

DESCRIPTION

showpack recursively find the last two versions of each file given as argument.
The files extensions can be omitted.
The working directory can be any directory inside a pack.
This command is helpful to find which file should be copied to the local source code directory to be modified.

DIAGNOSTICS

This command will not work if the current directory does not look like the inside of a pack. Then the following message would be displayed :
"The working directory looks neither like the main directory of a pack nor the inside of a pack. No show."
