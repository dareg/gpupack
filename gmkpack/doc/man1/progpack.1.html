NAME

progpack − display the programs which can be built by gmkpack

SYNOPSIS

progpack

DESCRIPTION

progpack displays the program names which can be built by gmkpack. These names are the arguments of the option -p of gmkpack.

FILES

Any user can create more executables for private usage. Then new executables definitions should be stored in the directory $HOME/.gmkpack/link following the pattern of the original exectables (see $GMKROOT/link ).
