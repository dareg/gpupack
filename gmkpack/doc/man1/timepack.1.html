NAME

timepack − display the most time consumming files of a pack at compile time

SYNOPSIS

timepack [ -n number-of-files ]

DESCRIPTION

timepack displays the most time consumming files of a pack at compile time. It supposes that the compilation as been performed with the variable ICS_TIMING_REPORT > 0

OPTIONS

-f number-of-files

How many of the most consumming files to display.
The default is to display the 20 most time consumming files

ENVIRONNEMENT

ROOTPACK

The default directory where to look for packs as an alternative to $HOME/rootpack

GMK_TIMEFILE_EXTENSION

The extension of the files containing the timing report. The default value is time
