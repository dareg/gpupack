NAME

usrpack − display user’s packs

SYNOPSIS

usrpack [ -h homepack ]

DESCRIPTION

usrpack displays the user existing packs names.

OPTIONS

-h homepack

Directory where to look for packs.
The default is to consider the directory $HOME/pack

ENVIRONNEMENT

HOMEPACK

The default directory where to look for packs as an alternative to $HOME/pack
