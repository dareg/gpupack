NAME

rmpack − remove packs

SYNOPSIS

rmpack [ -f ] pack1 [ pack2 ] ...

DESCRIPTION

rmpack removes the packs directories. If the executables directories are links outside the packs, then they are removed as well. If aliases have been created in the shell rc file, then these aliases are removed as well.

OPTIONS

-f

By default, rmpack asks for confirmation before running. Use this option to force the action without confirmation.

DIAGNOSTICS

This command will not work for an argument which does not look like a pack. Then the following message would be displayed :
"The directory target does not look like a pack. No remove."
