NAME

lspack − list all files within a pack.

SYNOPSIS

lspack [ dir1 ] [ dir2 ] ...

DESCRIPTION

lspack recursively lists all the files below the directories given as arguments, except hidden files.
If no directory is specified, then all the files from the current directory and below will be listed.

DIAGNOSTICS

If the working directory is lower than the local source directory then the pack content will not be fully listed.
