NAME

gmkfilemaker − configuration file setup assistant

SYNOPSIS

gmkfilemaker

DESCRIPTION

gmkfilemaker is an interactive procedure to assist the users making configuration files for gmkpack.
This command is expected to be run after the first installation of gmkpack and before making the first pack with it.
As "configure" would do, it tries to get informations from the system and ask the user for complementary informations.
This command can be run more than once, especially to run gmkpack with different compilers.

DIAGNOSTICS

After running this command, it is recommended to edit the resulting configuration file and make sure the setup looks correct. It is also an opportinity to improve the customisation of the configuration files.
