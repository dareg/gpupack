NAME

logpack − display the logfile of a pack.

SYNOPSIS

logpack

DESCRIPTION

logpack displays on the standard output the logfile of a pack creation. The working directory must be the pack main directory.
This command is helpful to control that the pack has been properly created.

DIAGNOSTICS

If the working directory is not a pack main directory or if the pack has been created with a version of gmkpack older than the version 5.1 then the following message is displayed :
"No gmkpack logfile in this directory."

logpack does not work for packs created with a version of gmkpack older than the version 5.1
