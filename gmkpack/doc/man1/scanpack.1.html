NAME

scanpack − scan the source code files within a pack.

SYNOPSIS

scanpack

DESCRIPTION

scanpack recursively lists from the current directory and below all the files which are compilable source files or include files as interpreted by gmkpack , that is all files with any of the following extensions will be listed :
"c" (C code), "F90", "F", "f90", "f" (fortran code), "h", "inc" (include files), "sql", "ddl" (odb98 code), "b" (bl95 code), "mnh", "MNH" (Meso-NH fortran code).

DIAGNOSTICS

If the working directory is lower than the local source directory then the pack content will not be fully listed.
Auto-generated interfaces, and more generally hidden files are not listed.
