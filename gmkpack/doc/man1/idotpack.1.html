NAME

idotpack − list all files with prefix "i." and suffix not ".L"

SYNOPSIS

idotpack

DESCRIPTION

idotpack recursively lists all the files with prefix "i." and suffix not ".L" in the current directory and below.
This command is dedicated to NEC SX platforms, where the pre-processed source files are usefull for debugging purpose. Such files are named as the source file but with prefix "i." ; the suffix ".L" corresponds to listings files.
