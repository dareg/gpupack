NAME

admpack − display administrator’s packs

SYNOPSIS

admpack [ -f rootpack ]

DESCRIPTION

admpack displays the administrator existing packs names.

OPTIONS

-f rootpack

Directory where to look for packs.
The default is to consider the directory $HOME/rootpack

ENVIRONNEMENT

ROOTPACK

The default directory where to look for packs as an alternative to $HOME/rootpack

AUTHOR

Ryad El Khatib <ryad.elkhatib@meteo.fr>

SEE ALSO
