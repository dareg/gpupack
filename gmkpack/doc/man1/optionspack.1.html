NAME

optionspack − display the available compilers configurations

SYNOPSIS

optionspack

DESCRIPTION

optionspack displays the configuration files names available, and for each of them, displays the equivalent pair of options -l and -o used by gmkpack.

AUTHOR

Ryad El Khatib <ryad.elkhatib@meteo.fr>
