Directory of reference packs

Before making a working pack you must have a ''reference'' pack already available, that is : with source code fully compiled and all object libraries made. Such packs should be located inside the directory ROOTPACK defined in your environment unless the option f of gmkpack is used to specifiy an alternative directory for the reference pack. For instance the this option can be used to specify that the reference pack is stored in your own HOMEPACK.
