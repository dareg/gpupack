Downloading the software

If GmkPack has already been installed by an administrator, you can skip this section.

You can get GmkPack from the following machines :

lxgmapxx:/public/gmap/khatib/install
cougar:/cnrm2_a/mrpm/mrpm602/gmkpack
merou:/home/mrpm602/public
hpce2.1:/hpce/ms_perm/hirald/tools
GmkPack comes as a tar-gzipped file of about 150 KBytes, named gmkpack.6.3.tgz or
gmkpack.6.3.tar.gz. This file must be unpacked in the target directory where GmkPack should be installed.
Example : 
% gunzip -c gmkpack.6.3.tgz | tar xvfo -C /home/khatib/bin
Then, before running GmkPack and as described in the following two sections, you have to :
  1. set some environment variables
  2. configure the software and build it
The complete installation should require between 2 and 15 MBytes, depending of the configuration and the machine.
