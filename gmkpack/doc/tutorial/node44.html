Listings

You can ask GmkPack to save output listings of the compilation for each subroutine or module : 

# Do you want to output the compilation listings ? (yes/no)
# ----------------------------------------------
export ICS_LIST=yes
Notice :
  • The default value is ''yes'' for ordinary packs ; The default value is ''no'' for main packs.
  • The output listings will be saved in the same directory as the corresponding source file and object file
  • The compiler options which condition the output listing file are defined in the configuration file.
  • If output the compilation listings are not requested a short report should appear in the general compilation logfile in case of errors.
