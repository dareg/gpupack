progpack

progpack displays the program names for which GmkPack can generate a compilation script. These names are the arguments of the option -p of gmkpack. Example : 
% progpack
adbias   atstprog      ctrodb    ddht      lamflag_odb   qscat_25to50
adscan   bator         ctroulan  eatstpro  lecbdap       qscat_bufr
aladin   biascv        cybias    etestadj  mandalay      qscat_dconeqc
aldodb   blend         cyprep    fcqodb    odb_compress  qscat_filter
arome    blendsur      cyscan    gobptout  odbtools      sptogp
aromodb  cabias        cysele    gptosp    pregpssol     testadj
arpege   cascan        ddhc      hcat      procadre      testfa
arpodb   check_limits  ddhr      ioassign  progrid
