Load map

You can ask GmkPack to produce a load map of each executable at linking time : 

# Do you want to output the load map(s) ? (yes/no)
# -------------------------------------
export ICS_MAP=yes
Notice :
  • The default value is ''no'' for all packs.
  • The load map can help debugging when an executable cannot be linked or when the behaviour of an executable is odd.
