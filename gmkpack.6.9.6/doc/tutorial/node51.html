clearpack

clearpack recursively removes all the files which are neither object files nor ''source files'' to GmkPack ; in other words, all files except the ones which have any of the following extension will be removed : o F90 F f90 f h inc sql c ddl mnh odb98.flags l y b. Notice :

  • Hidden files will not be removed.
  • The working directory can be $HOMEPACK/src/local or any of its subdirectory. If the working directory is lower than the local source directory then the pack may not be completely ''cleared''. If the working directory is higher than the local source directory then the pack will not be ''cleared''.
