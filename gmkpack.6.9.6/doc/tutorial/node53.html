scanpack

scanpack recursively list all the files which are ''source files'' to GmkPack ; in other words, all the files which have any of the following extension will be listed : F90 F f90 f h inc sql c ddl mnh odb98.flags l y b. Notice :

  • Hidden files will not be listed.
  • The working directory should preferably be $HOMEPACK/src/local to list the full content of a pack.
